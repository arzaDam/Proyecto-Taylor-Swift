## ✨ Features

### 🎨 Diseño

* Fondo rotativo con fade cada 6s
* Glassmorphism moderno
* Responsive (mobile-first)
* Tipografías elegantes y branding consistente

### 🎵 Reproductor

* Player flotante
* Playlist dinámica
* Barra de progreso y volumen

### 📄 Páginas

* Inicio (landing + stats)
* Fan Club (eras)
* Store (productos)
* Tours (fechas + galería)
* Contacto validado
* Fans (interacción + chat)

### 🔒 Seguridad

* Sanitización XSS
* Validación de formularios
* Rate limiting
* Token CSRF

### ♿ Accesibilidad

* ARIA labels
* Navegación por teclado
* UI usable en mobile

### 🔍 SEO

* Meta tags completos
* Open Graph + Twitter Card
* JSON-LD Schema

### 🤖 Interactividad

* Chat tipo IA
* Likes en fotos
* Subida de imágenes
* Lightbox
